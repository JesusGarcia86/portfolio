El proyecto de los deseos

El proyecto de los deseos es el que hice para obtener mi liston negro(que es el mayor logro de Coding Dojo) y mi graduación en Python y fue creado con Django. Basicamente permite al usuario iniciar sesion o registrarse. Todo tiene sus propias validaciones para asegurar que el usuario ingrese la información correcta, las contraseñas son encriptadas con bcrypt para maximizar la seguridad del usuario. Una vez iniciada la sesión los usuarios puede ingresar o editar sus deseos para que otros usuarios esten informados de ello, una vez que el deseo es concedido, el usuario simplement da click en el link de granted y automaticamente se transfiere a la lista de deseos concedidos.

Holiday Greetings

Para este projecto colabore con 3 programadores de software para crear desde la planificacion hasta la ejecucion. Este programa se construyo para permitir a los usuarios estar en contacto con sus seres amados enviandoles una tarjeta virtual. Permite al usuario iniciar sesion o registrarse. Todo tiene sus propias validaciones para asegurar que el usuario ingrese la información correcta, las contraseñas son encriptadas con bcrypt para maximizar la seguridad del usuario. Una vez iniciada sesion los usuarios pueden crear su tarjeta virtual ya sea escogiendo de las que tenemos en la base de datos o pueden cargar su propia imagen o video, tambien pueden agregar notas personalizadas y musica, una vez que terminar de crear su tarjeta pueden mandarla directamente a una o mas personas. Las herramientas usadas para este proyecto fueron HTML, CSS, Javascript y algunos APIs externos.
